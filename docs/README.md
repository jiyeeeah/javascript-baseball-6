## ⚾ Baseball 기능 구현 목록

### 📌 게임 시작

- [x] 게임 시작 문구 **출력**
- [x] 컴퓨터의 숫자 생성 기능
  - [x] 1 ~ 9까지 중 겹치지 않는 3개의 숫자

### 📌 게임 진행

- [x] 사용자 **입력**
  - [x] 입력 받은 값의 validation 확인
  - [x] 문자가 아닌 '숫자'인지
  - [x] 3자리의 수인지
  - [x] [ERROR] 입력값이 유효하지 않은 경우, throw 문으로 종료
- [x] 볼, 스트라이크, 낫싱 결과 문구 **출력**
- [x] 맞추면 게임 종료 문구 출력

### 📌 게임 종료

- [x] 게임 종료 후, 재시작 여부 물어보는 입력 받기
  - [x] 1이나 2를 받았는지 validation 확인
- [x] 입력값이 1일 경우, 게임 재시작
